Linux Command

  • find
$ find [path] -name [name]
  • !
$ !! # 上一条命令
$ !-x # 上x条命令
$ !!:s/p1/p2/ # 将上条命令的p1替换成p2
$ ^p1^p2^ # 等同于上条命令
  • {}
$ echo {a,b}{a,b} # 会生成全排列的aa ab ba bb
$ cp filename{,.bak} # 备份文件
  • >
$ > file.txx # 清空或创建文件
  • ssh
$ ssh-keygen # 生成公钥
$ ssh-copy-id remote-machine # 直接把本机的公钥写到远程主机的~/.ssh/authorized_keys内
